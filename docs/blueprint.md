# **App Name**: ScamSensei

## Core Features:

- Instant Scam Assessment: AI-powered message analysis tool to identify potential scams, providing a risk assessment: "looks safe," "be careful," or "likely a scam". Explains reasoning with real-world scam connections.
- The Coach Approach: Interactive guided conversation tool. Uses AI to examine message attributes (sender, language, links, requests) in a friendly manner, posing simple questions, assisting the user in self-discovery of potential scams.
- Multi-Input Message Capture: Easy copy-paste, type-in or photo import of suspicious messages.
- Readable Results: Display analyzed text in large, clear font.
- Support Resources: Provides helpful phone numbers and resource links for reporting scams.
- Share with Family: Offer pre-made text to allow the user to easily forward results to family members.
- Practice Mode: Test the scam-detection skills of users with a safe 'quiz' mode that contains up-to-date examples of potential scams.

## Style Guidelines:

- Primary color: HSL 48, 75%, 50% - A vibrant gold (#E0B30D) symbolizing knowledge, awareness, and reliability in detecting scams.
- Background color: HSL 48, 20%, 95% - A very light tint of pale gold (#FDFBF4) to allow a calm user experience.
- Accent color: HSL 18, 65%, 50% - A warm orange (#E07A0D) to highlight calls to action, danger warnings, or key information.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for both headlines and body text.
- Simple, clear icons to represent different message sources (email, text, phone) and result types (safe, warning, scam).
- Clean, spacious layout to avoid overwhelming the user with information. Prioritize readability and ease of navigation.
- Subtle transitions and animations to guide the user through the analysis process without distracting from the content.
